What Changed (v4.1 → v5)

#ChangeBefore (v4.1)After (v5)
NEW CoinGlass Confidence Modifier CG data polled but NEVER used in trade decisions 6 signals feed into a ±0.50 confidence modifier that confirms or blocks trades
NEW Taker Buy/Sell Volume Not tracked 1-min taker aggression — who is pushing the market
NEW Smart Money Divergence No positioning data Top traders vs crowd — strongest contrarian signal
FIX CoinGlass Intervals 4-hour candles (useless for 5-min windows) 1-minute candles (Standard plan unlocked)
SAME Core Regime Logic VPIN + Delta remain primary. CoinGlass is CONFIRMING only.
Key Principle: CoinGlass does NOT change the direction or regime classification. It only adjusts confidence after VPIN + delta have already decided. This means v5 is strictly backward-compatible — if CG is disconnected, it works identically to v4.1.
